```text
.
├── triangle.py

└── README.md
```
## Функционал

triangle.py

Два основных метода:

    triangle_area(a, b, c) — вычисляет площадь по формуле Герона

    triangle_perimeter(a, b, c) — вычисляет периметр (сумма сторон)

## Встроенная валидация

Функции проверяют:

    - Аргументы — числа (int или float)

    - Стороны строго положительны (> 0)

    - Выполнено неравенство треугольника: a+b>ca+b>c, a+c>ba+c>b, b+c>ab+c>a

При нарушении условий выбрасываются исключения:

    TypeError — если аргумент не число

    ValueError — если значение некорректно

## Тесты (tests.py)

Всего 8 тестов в классе TriangleTestCase:
Позитивные тесты (4 шт.)

Проверяют корректную работу на валидных данных:

    test_equilateral_area — площадь равностороннего треугольника (3, 3, 3)

    test_equilateral_perimeter — периметр равностороннего треугольника

    test_isosceles_area — площадь равнобедренного треугольника (3, 3, 2)

    test_isosceles_perimeter — периметр равнобедренного треугольника

Негативные тесты (4 шт.)

Проверяют корректный выброс ошибок:

    test_negative_side_raises — ошибка на отрицательной стороне

    test_zero_side_raises — ошибка на нулевой стороне

    test_triangle_inequality_raises — ошибка при нарушении неравенства

    test_type_error_raises — ошибка на неправильном типе аргумента

## Запуск

```bash
python -m unittest tests.py
```
